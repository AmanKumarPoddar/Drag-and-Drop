## DragTrack - A Kanban Board for Drag and Drop

DragTrack is a simple Kanban board that allows you to drag and drop tasks between columns. It is built using:

- React
- DnD-Kit
- TypeScript
- Tailwind CSS
- Framer-Motion
- Lucide Icons (for the icons)
